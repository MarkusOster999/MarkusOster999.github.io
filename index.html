ТЕОРИЯ ГРАФОВ

  • граф

    • Граф - абстрактный математический объект, представляющий собой множество вершин графа и набор рёбер, то есть соединений между парами вершин.

    Связанный граф - граф все вершины которого имеют связь (можно через другие вершины).

    Ориентированый граф - граф рёбра которого имеют напрвленносить.

    Степень вершины графа - количество рёбер исходящик из данной вершины.

    Дерево - связанный граф не содержащий замкнутых элементов.

    Изоморфизм графов - два графа называются изоморфными если они имеют равное число рёбер, равное число вершин, и соответствующие рёбра одного графа соединяют соответственные вершины второго графа.

    Эйлеров цикл - цикл(замкнутый элемент графа), содержащий в себе все рёбра графа (вершины могут повторяться).

    Гамильтонов цикл - простой цикл в графе, содержащий все вершины графа ровно по одному разу.

    Кратчайший путь алгоритмом дейкстры - алгоритм поиска кратчайшего пути между двумя любыми вершинами с учётом веса рёбер путём присваивания вершинам веса их минимального пути дор данной точки
